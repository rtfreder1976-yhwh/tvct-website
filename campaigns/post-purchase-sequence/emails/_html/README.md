# Paste-ready HTML for GHL — Post-Purchase Sequence

Each `.html` file below is a fully self-contained brand-styled email body. Paste each into GHL's email builder using **Marketing → Emails → Templates → + New → Import HTML**.

After pasting, set the template name, subject line, and preview text from this table:

| # | GHL template name | Subject (variant A) | Preview text |
| - | - | - | - |
| 1 | `TVCT — Post-Purchase #1 — Booking Confirmed` | You're on the schedule — here's what happens next | A short note about clean day, the photos, and what to do with the dog. |
| 2 | `TVCT — Post-Purchase #2 — Day Of Clean` | We're on the way today (and a quick favor) | Names of who's coming, what time, and one thing you can do for us later. |
| 3 | `TVCT — Post-Purchase #3 — Post Clean Followup` | How did we do? (and your photo proof, attached) | Two-minute answer. Honest one is the most useful. |
| 4 | `TVCT — Post-Purchase #4 — Recurring Upsell` | Most customers book us again around now | Bi-weekly is the right answer for ~70% of our customers. Here's how to tell if you're one of them. |
