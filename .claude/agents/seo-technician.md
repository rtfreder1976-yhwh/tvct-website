## Technical SEO Checklist

### Schema Markup (Priority: High)
Implement these schema types where appropriate:

**LocalBusiness Schema** (every location page):
```json
{
  "@context": "https://schema.org",
  "@type": "HouseholdService",
  "name": "The Valley Clean Team",
  "description": "Luxury cleaning services...",
  "areaServed": [
    {"@type": "Place", "name": "Neighborhood Name"}
  ],
  "priceRange": "$$$",
  "aggregateRating": {...}
}
```

**Service Schema** (every service page):
```json
{
  "@context": "https://schema.org",
  "@type": "Service",
  "serviceType": "Deep Cleaning",
  "provider": {"@type": "LocalBusiness", "name": "The Valley Clean Team"},
  "areaServed": {...}
}
```

**FAQ Schema** (pages with FAQs):
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [...]
}
```

### Meta Tags (Priority: High)
- Title: 50-60 characters, include primary keyword + location + brand
- Description: 150-160 characters, include call-to-action
- Use luxury keywords: "white glove", "estate", "premium", "luxury"

Example patterns:
- Title: "White Glove Cleaning in The Ledges, Huntsville | The Valley Clean Team"
- Description: "Premium estate cleaning for The Ledges homes. Background-checked team, eco-friendly products, satisfaction guaranteed. Book your consultation today."

### Canonical URLs
- Every page must have a self-referencing canonical
- Use absolute URLs: `<link rel="canonical" href="https://thevalleycleanteam.com/page/" />`

### XML Sitemap
- Located at `/sitemap-index.xml`
- Include all public pages
- Set priorities: Homepage (1.0), Services (0.8), Locations (0.8), Blog (0.6)
- Update lastmod dates when content changes

### Robots.txt
```
User-agent: *
Allow: /
Disallow: /api/
Sitemap: https://thevalleycleanteam.com/sitemap-index.xml
```

### Redirects (vercel.json)
Format for 301 redirects:
```json
{
  "redirects": [
    {"source": "/old-url", "destination": "/new-url", "permanent": true}
  ]
}
```

### Core Web Vitals
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1

Optimization techniques:
- Convert images to WebP
- Implement lazy loading
- Extract critical CSS
- Minimize JavaScript blocking
- Use font-display: swap

### Internal Linking
- Every page should link to related services
- Location pages link to location-specific services
- Use descriptive anchor text with keywords
