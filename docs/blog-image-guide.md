# Blog Image Guide & Checklist

## Quick Reference
- **Total images needed:** 30 (one per blog post)
- **Dimensions:** 1200 x 630px (optimal for social sharing + blog display)
- **File format:** .jpg or .webp (for faster loading)
- **Location:** `/public/images/blog/`

---

## Canva Template Setup

### Create ONE Master Template (1200 x 630px)

**Layout Recommendation:**
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  [Background Photo - 60% opacity dark overlay]          │
│                                                         │
│     ┌─────────────────────────────────────────┐         │
│     │  HEADLINE TEXT                          │         │
│     │  (Large, white, bold)                   │         │
│     └─────────────────────────────────────────┘         │
│                                                         │
│     ┌─────────────────────────────────────────┐         │
│     │  Location Tag                           │         │
│     │  (Smaller, accent color box)            │         │
│     └─────────────────────────────────────────┘         │
│                                                         │
│                              [TVCT Logo - bottom right] │
└─────────────────────────────────────────────────────────┘
```

**Brand Elements:**
- Primary accent color: #FFA985 (peach/coral from your site)
- Font: Match your website (or use Montserrat/Poppins as alternatives)
- Logo: Place in consistent position (bottom right recommended)
- Overlay: Dark gradient (black at 40-60% opacity) for text readability

---

## Naming Convention

Format: `[topic]-[location].jpg`

Examples:
- `reliable-cleaning-florence-al.jpg`
- `house-cleaning-cost-huntsville-al.jpg`
- `commercial-cleaning-nashville-tn.jpg`

## Workflow Summary

### One-Time Setup (30 min):
1. Create Canva account (free)
2. Create new design: 1200 x 630px
3. Add dark overlay layer
4. Add text boxes (headline + location tag)
5. Add TVCT logo
6. Save as template

### Per Image (5 min each):
1. Duplicate template
2. Swap background photo
3. Update headline text
4. Update location tag
5. Export as JPG
6. Rename file per naming convention
7. Upload to `/public/images/blog/`

### Total Time Estimate:
- Setup: 30 minutes
- 30 images × 5 min = 150 minutes (2.5 hours)
- **Total: ~3 hours**
